# Desafio Frontend

## Setup
      
1. Instalação do [NodeJs](https://nodejs.org/pt-br/download/)
2. Instação do [Yarn](https://classic.yarnpkg.com/pt-BR/docs/install/#windows-stable) 
3. Comando `yarn` na pasta clonada


Para rodar o projeto `yarn start`

O projeto conta com uma *roteirização* interna por meio do ```react-router-dom```, apesar de ter apenas uma rota

O JSON é importado para a aplicação por meio de import do REACT

O Css está em formato Sass

Foi ultlizado uma componentização básica para os cards de anúncios e página incial
